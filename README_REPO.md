# North Star Build 2.0

This repository contains the North Star Build governed autonomy platform, its CLI, and the supporting framework documents.

## Repo Layout

```
NorthStarBuild_2.0/
├── packages/      # Core libraries + CLI
├── apps/          # Docs + marketing site
├── examples/      # Sample governed projects
├── docs/          # Strategy, blueprint, and research docs
├── BRIDGE.md
├── NORTH_STAR_BOOTSTRAP.md
├── north-star-blueprint/
└── master-build-framework/
```

## Quick Start

```bash
npm i -g @nsb/cli
nsb init --profile professional --tools claude,cursor,codex
```
